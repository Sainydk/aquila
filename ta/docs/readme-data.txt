Базовая единица информации - объекты типа Value, обеспечивающие доступ к
текущему и историческим значениям параметра технического анализа. Объекты Value
предусматривают генерацию события изменения значения, что позволяет
организовывать расчет других значений при поступлении новых данных. Таким
образом, различные Value могут быть связаны между собой произвольным образом.
При этом, Value-источники данных ничего не знаю о Value-потребителях данных,
так же как и о других несвязанных Value-объектах.

Объекты Value могут быть сгруппированы с помощью контейнере ValueContainer.
Помимо группировки, контейнер не выполняет какой-либо обработки данных.
Контейнеры могут использоваться например как точка доступа к индикатором,
расчитываемых в едином таймфрейме.

Для расчета значений используются объекты типа ValueSource. Объекты этого типа
реализуют специфические алгоритмы расчета и механизмы извлечения данных из
других источников.
